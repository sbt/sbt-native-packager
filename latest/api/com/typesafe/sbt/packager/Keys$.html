com.typesafe.sbt.packager

Keys 

object Keys extends NativePackagerKeys with UniversalKeys with LinuxKeys with WindowsKeys with DockerKeys with DebianKeys with RpmKeys with JavaAppKeys with JavaServerAppKeys with SystemloaderKeys with BashStartScriptKeys with BatStartScriptKeys with ValidationKeys

This Keys object can be used for

  • non autoplugin builds
  • import single keys, which are not inside the autoImport

Non autoplugin builds 

import com.typesafe.sbt.packager.Keys._

packageName := ""

autoplugin builds 

NativePackagerKeys.packageName := ""
